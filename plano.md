# Gerenciamento de Notas

Cadastrar Aluno: 
Aqui nesta secao, o sistema devera informar o nome do aluno, o numero de matricula, a turma que ele esta Matriculado e o Curso que ele esta fazendo Apos isso o Sistema devera exibir uma mensagem dizendo que o aluno foi Cadastrado Com Sucesso


Editar Aluno:
    Caso queira editar o nome de algum algum aluno, o sistema devera permitir a edicao. Mas para isso, o sistema devera informar o numero de matricula
    Depois mostrar uma mensagem dizendo que o aluno foi editado com sucesso.

Cadastrar Disciplina: 
Nesta secao para Cadastrar uma determinada disciplina o Sistema devera perguntar o Nome da disciplina que eu desejo Cadastrar, O nome do Professor que ira Ministrar aquela disciplina e a carga horaria da disciplina, por exemplo 40 horas, 120 horas, 1200 horas dependendo da quantidade de Conteudo que sera Ministrado nesta disciplina. Depois diso tudo, o Sistema devera informar uma mensagem dizendo que a Disciplina foi Cadastrada com Sucesso


Editar Disciplina:
    Caso queira editar o nome de alguma disciplina, o sistema devera permitir a edicao.
    Depois mostrar uma mensagem dizendo que a disciplina foi editada com sucesso.


Lancar Nota:
    O sistema devera informar as notas que o Aluno obteve ao longo do semestre. Exemplo 1 semestre 8, 2 semestre 7, etc...

Adicionar atividades:
    Nesta parte, o sistema devera permitir adicionar atividades, como as provas dos alunos, ponto extra, Trabalhos Academicos
    Depois que tiver com as atividades prontas para adicionar.  O sistema devera exibir uma mensagem dizendo que a Atividade foi Adicionada com Sucesso.


Verificar Situacao do Aluno:
    Aqui para ver se o Aluno foi Aprovado o sistema devera fazer o seguinte Calculo:

        Media = (Trabalhos + atividades + provas academicas + ponto extra) /3

        Todo esse calculo devera dar media 6:

        Se a Media for maior que 6, o Sistema devera exibir uma mensagem dizendo que o Aluno foi Aprovado!
        Se for exatamente 6, o  sistema devera informar que o Aluno Passou Arrastado, mas Passou

        Agora se a Media for menor que 6, o sistema devera dizer que o Aluno foi REPROVADO e devera refazer tudo de novo


       Verificar historico do Aluno:
        O sistema devera informar o historico do Aluno, Contendo nome, matricula, disciplina, curso, turma, as notas e a situacao do aluno
