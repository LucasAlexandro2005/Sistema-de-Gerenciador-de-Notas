# Sistema de Controle Financeiro

# Menu:
1- Cadastrar Produto
2- Listar Produtos
3- Registrar Entrada
4- Registrar Saida
5-Cadastrar Cliente
6-Registrar Cliente
7- Relatorios
8- Sair

Cadastrar Produto:
    Usuario Cadastra um Produto
    Sistema Salva o Produto
    Ao escolher um Produto, O sistema devera:
    Mostrar: Produto Cadastrado com Sucesso


 Listar Produtos:
 Ao Cadastrar um Produto, O sistema deve mostrar quais produtos estao Listados depois de Cadastrados

 Registrar Entrada:
    O Sistema devera registrar a entrada de Cada Produto

 Registrar Saida:
    O Sistema devera registrar a saida de Cada Produto   

 Cadastrar Cliente:
    Nesta secao, o sistema devera perguntar qual Cliente deseja Cadastrar incluindo o Nome do Cliente e o CPF
    Apos Cadastrar o Cliente. O sistema devera dizer que o Cliente foi Cadastrado com Sucesso   


Registrar Cliente:
    Aqui nesta Parte, apos o cadastro do Cliente, o Sistema devera registrar o cliente Cadastrado
    Apos isso o sistema devera dizer que o Cliente foi Registrado com Sucesso

Relatorios:
Nesta Parte, o sistema devera trazer um relatorio completo de Cada Produto



Sair:
    Aqui, o sistema devera perguntar o usuario se ele quer continuar usando ou se ele quer sair
    se o Usuario continuar usando o sistema
    Mostrar: Sistema em Andamento
    Senao:
        Mostrar: Sistema finalizado com Sucesso
